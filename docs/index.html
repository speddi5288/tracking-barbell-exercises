This end-to-end data science pipeline takes raw CSV output from a MetaWear wearable sensor and produces a trained classifier that can identify which barbell exercise a lifter is performing — and count how many reps they have done.

# Pipeline

1

Data Collection & Merging

187 CSV files (accelerometer 12.5 Hz + gyroscope 25 Hz) loaded, labeled from filenames, merged, and resampled to a uniform 200ms interval.

make_dataset.py
2

Outlier Removal

Three methods compared (IQR, Chauvenet's Criterion, LOF). Chauvenet's Criterion selected for per-label robustness.

remove_outliers.py
3

Feature Engineering

Butterworth low-pass filter → PCA → magnitude features → rolling statistics → FFT frequency features → K-Means cluster labels.

build_features.py
4

Model Training

6 classifiers × 5 feature sets evaluated with grid search. Participant-based split tests real-world generalization.

train_model.py
5

Rep Counting

Exercise-specific low-pass filter + peak detection on acceleration or gyroscope magnitude, tuned per movement pattern.

count_repetitions.py

# Feature Engineering Detail

Butterworth Low-Pass Filter

Cutoff: 1.2 Hz, order 5. Zero-phase filtering (filtfilt) eliminates timing artifacts. Removes high-frequency noise while preserving exercise motion patterns.

Principal Component Analysis

6 raw sensor axes reduced to 3 orthogonal principal components, capturing maximum variance while removing redundancy between accelerometer axes.

Magnitude Features

acc_r = √(x² + y² + z²)
gyr_r = √(x² + y² + z²)
Rotation-invariant measures of overall movement intensity.

Temporal Abstraction

Rolling window (size = 5 samples = 1 second) computes mean and standard deviation for each feature — capturing movement rhythm over time.

FFT Frequency Features

Window = 14 samples (~2.8s, matching average rep duration). Extracts per-frequency amplitudes, weighted mean frequency, and power spectral entropy.

K-Means Clustering

5 clusters fitted on acceleration data. Cluster label added as a categorical feature, providing a data-driven motion signature grouping.
